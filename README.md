# Patchouli OS

This repository is about a simple OS :)

I was bored but it works lol.

## Requirements

ArchLinux :

```bash
sudo pacman -Syy
sudo pacman -S nasm qemu gcc 
```

Debian :

```bash
sudo apt update
sudo apt install nasm qemu-system-x86 gcc
```

## Build and run

```bash
make run
```

## Just build

```bash
make all
```

## Cleaning

```bash
make clean
```
